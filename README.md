#  Extreme

**Extreme** is a powerful and modern DLL or for Windows. It supports advanced ion methods including Manual Mapping, Stealth ion, and more, making it suitable for game modding and development.

## Key Features
- Multiple ion methods (Manual Map, LoadLibrary, etc.)
- Stealth ion support
- x64 and x86 process compatibility
- Easy process selection with filtering
- One-click DLL ion
- Lightweight and portable

## Getting Started
1. **Download** the latest version using the button above.
2. **Extract** the archive.
3. **Run** `Extremeor.exe` as Administrator.
4. **Select target process** from the list.
5. **Browse and select** your DLL file.
6. Click ****.
